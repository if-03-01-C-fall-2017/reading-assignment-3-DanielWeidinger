Lists and Tables



Lists

Every list consists of a list tag (ul = unordered list, ol = ordered list)
and the list items(tag: li) as nested tags

unordered list:


example fruits:
  • apple
  • orange
  • pears


ordered list:


example fruits:
  1. apple
  2. orange
  3. pears

Tables

Tables are defined with the table tag Each table row is defined with the tr tag.
A table header is defined with the th tag. By default, table headings are bold and centered.
A table data/cell is defined with the td tag.

example pupils:

firstname lastname number
Daniel Weidinger 1
Max Mustermann 2
